# MarQet Backend

Node.js + Express.js backend for the MarQet hyperlocal commerce platform.

## Features
- Node.js + Express.js
- MongoDB + Mongoose
- JWT authentication
- Socket.io for real-time
- bcryptjs for password hashing

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Setup Environment variables:
   Copy `.env.example` to `.env` and fill in the values.

3. Run the development server:
   ```bash
   npm run dev
   ```
